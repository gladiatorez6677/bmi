# PT Bajeng Maritime Indonesia — Company Profile

Website company profile statis untuk **PT Bajeng Maritime Indonesia (BMI)**.

- **Domain:** https://bajengmaritime.co.id
- **Bidang usaha:** Angkutan Laut (barang umum & khusus), Angkutan Darat, Pergudangan & Penyimpanan, Perdagangan Besar Energi/BBM, Logam & Komoditas.
- **Teknologi:** HTML, CSS, JavaScript murni (tanpa build step) — cukup dilayani sebagai file statis.

## Struktur
```
index.html        Halaman utama (single page)
css/style.css     Tema & tata letak (biru maritim + aksen merah)
js/main.js        Interaksi (nav, reveal, count-up, form)
assets/logo.png   Logo BMI
```

## Menjalankan lokal
Cukup buka `index.html` di browser, atau: `python3 -m http.server`.
